# VoiceTasker

VoiceTasker is a Python-based voice-enabled task manager.  
It allows you to add tasks via text or voice, and hear them spoken back using offline text-to-speech.

## Features
- Add tasks manually or by voice
- Store tasks in SQLite database
- Speak tasks aloud using pyttsx3
- Simple Flask web dashboard

## Tech Stack
- Python
- Flask
- SpeechRecognition
- pyttsx3
- SQLite
